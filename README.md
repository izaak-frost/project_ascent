# 🧗 Ascent: Expert Mode

A modern expert-style Minecraft modpack built around progression, difficulty, and mastery.

Inspired by packs like FTB Infinity Evolved: Expert Mode, Ascent reimagines that experience for modern Minecraft using NeoForge, introducing primitive survival, gated progression, and long-term automation challenges.

As an avid modded Minecraft player, I found myself repeatedly falling into the same pattern: rushing powerful mechanics, skipping progression, and trivialising the game far too quickly. Ascent is a response to that.

It’s designed to slow things down, make every upgrade feel earned, and push you to engage with systems rather than bypass them.

#### Progress isn’t given, it’s built.


## 🌍 Vision

Ascent is designed around a simple idea:

> **You start with nothing — and earn everything.**

Progression is intentionally slowed and structured to create:

* Meaningful early-game survival
* Clear technological milestones
* Deep, rewarding late-game systems

This is not a kitchen-sink pack — it is a **curated journey**.


## 🔥 Core Features

### 🪨 Primitive Progression

* Logs require **axes to harvest**
* Stone is broken into **stone chunks** before becoming usable
* Early tools are crafted from **flint and basic materials**
* Resource acquisition is **manual and intentional**

---

### ⚙️ Expert Crafting & Gating

* Recipes are rebalanced for **progression and challenge**
* Materials flow through multiple stages before use
* Mods are integrated into a **cohesive progression path**

---

### 🧠 Thoughtful Difficulty

* Deliberately grindy
* Rewards planning and system-building
* Encourages automation and optimisation

---

### 🧰 Custom Systems (KubeJS)

* Fully custom progression logic
* Custom items, recipes, and interactions
* Modular script structure for maintainability

---

## 🏗️ Technical Overview

* **Minecraft Version:** 1.21.1
* **Mod Loader:** NeoForge
* **Scripting:** KubeJS + LootJS
* **Design Philosophy:** Modular, maintainable, extensible


## 📁 Project Structure

```text
kubejs/
  startup_scripts/   # Item registration, tool tiers
  server_scripts/    # Recipes, loot, progression logic
  client_scripts/    # UI, tooltips (future)

config/              # Mod configuration
defaultconfigs/      # Default server configs
```



## 🚀 Getting Started

1. Install via your preferred launcher (Prism, CurseForge, etc.)
2. Ensure correct **NeoForge version**
3. Launch the game
4. Start a new world
